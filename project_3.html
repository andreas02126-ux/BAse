프로젝트 개요

이 프로젝트는 테스트 용도로 작성된 포트폴리오 예시입니다.

역할 및 기여

  • 기획 및 설계
  • 프론트엔드 개발
  • UI/UX 디자인

주요 성과

  • 사용자 친화적인 인터페이스 제공
  • 모바일 반응형 레이아웃 적용
  • 효율적인 코드 구조 설계

사용 기술/도구

  • HTML5, CSS3
  • JavaScript (ES6)
  • Figma (UI 설계)

개선점 및 배운 점

테스트 프로젝트였지만, 구조화된 설계와 효율적인 개발 워크플로우의 중요성을 배웠습니다.
